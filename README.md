# JQ-BuyList-exercise 
![image](https://img.shields.io/badge/jQuery-exercise-brightgreen.svg)

- replace() 字串替換
- append() 插入選定元素裡的最未端
- attr() 抓取屬性
- val() 讀取選定元素的內容
- trim() 去除字串前後的空白
- push() 在陣列最後方新增元素，並傳回且插入
- splice() 為一個陣列刪除並且新增陣列元素
